# GGTI AI 原型提示词

## 版本一：适合生成高保真网页原型

设计一个面向中国年轻游戏玩家的网页产品原型，产品名称是 **GGTI**，核心主题是“发现你的游戏人格”。

这是一个 **游戏人格测试网站**，用户通过回答少量问题，获得自己的游戏人格类型、团队角色倾向、决策风格和潜在优势。整体产品不是传统问卷网站，而是一个带有 **游戏氛围、未来感、沉浸感、轻科幻感** 的互动体验产品。

### 整体设计风格
- 深色背景为主，主色偏黑蓝、深紫、靛蓝
- 页面带有柔和的紫蓝渐变光晕、模糊背景光球、细网格纹理、轻微噪点覆盖
- UI 风格偏 **玻璃拟态 + 发光按钮 + 未来感卡片**
- 视觉气质参考：电竞感、赛博感、游戏启动器、次世代测试产品页面
- 不要做成企业官网或传统教育问卷页面，要更酷、更沉浸、更像游戏相关产品
- 页面元素圆角偏大，阴影柔和，层次清晰
- 有轻微悬浮动效与微交互感，但整体不杂乱

### 页面结构
请输出一个完整桌面端网页原型，包含以下 4 个页面：

#### 1. 首页 Landing Page
目标：激发兴趣，让用户立即开始测试。

页面内容：
- 顶部固定导航栏：Logo「GGTI」，导航项包括：首页、测试、关于我们、帮助
- 首屏 Hero 区：
  - 大标题：发现你的游戏人格
  - 副标题：通过 7 道专业题目，解析你在游戏世界中的独特性格、团队价值和决策风格
  - 一个非常醒目的主按钮：开始测试
  - 可加入“10,000+ 玩家已测试”之类的社交证明徽标
- 首页下半部分为 3 个玻璃风特性卡片：
  - 快速选择
  - 轻量流畅
  - 可扩展题库
- 页面底部有简洁 футер，强调测试流程：选择游戏 → 回答问题 → 解锁结果

#### 2. 选择游戏页
目标：让用户先从熟悉的游戏世界进入测试，增强代入感。

页面内容：
- 页面标题：选择游戏
- 副标题：先选一个入口，再进入题目页
- 中间区域展示 2 到 4 张竖版游戏海报卡片
- 每张卡片都像电影海报或游戏封面：
  - 大图占主体
  - 底部叠加游戏名称
  - 可有一句简短说明
  - hover 时轻微上浮、放大、发光
- 页面整体不要做成海报墙，而是保留适度留白，让每张卡片更有存在感
- 底部有返回首页按钮

#### 3. 测试题目页
目标：降低答题压力，让用户专注于当前问题。

页面内容：
- 顶部标题：题目页 / 开始测试
- 提供返回选择游戏入口
- 页面上方有进度条，显示例如：问题 1 / 7，14%
- 中间是一个大号玻璃卡片，里面放一个问题
- 每次只展示 1 道题目
- 题目下方是继续按钮
- 页面可以附加重新选择游戏、返回首页等辅助操作
- 视觉上要简洁，减少干扰，突出题目本身

#### 4. 结果页
目标：让结果像一种身份解锁，强化完成测试后的奖励感。

页面内容：
- 大型结果展示区
- 可展示人格类型名称，例如：战场指挥官 / 自由探索者 / 团队粘合剂 / 战术策略家
- 中间有一个象征性的徽章、图标或角色插画占位
- 下方有一段结果解释：说明用户在游戏中的行为风格、团队作用、偏好和潜力
- 再下方可展示：优势标签、适合的玩法、建议方向
- 页面底部有按钮：重新测试、返回首页、分享结果
- 结果页要有“解锁身份”的仪式感和高级感

### 版式要求
- 优先输出桌面端高保真网页原型
- 保持统一的视觉系统：深色背景、玻璃卡片、紫蓝发光、圆角大卡片
- 内容排版现代、克制、整洁
- 不要堆太多文案，重点突出标题、行动按钮、卡片和结果
- 整体感觉应该像一个高品质游戏人格测试产品，而不是普通表单网站

### 关键词
游戏人格测试、沉浸式、深色未来感、玻璃拟态、紫蓝霓虹、游戏海报、互动网页、结果解锁、电竞氛围、次世代 UI

## 版本二：适合给 AI 生成线框图 / 产品原型草图

请为一个名为 **GGTI** 的网页产品生成原型图。

产品定位：
这是一个面向游戏玩家的“游戏人格测试”网站。用户先选择自己熟悉的游戏，再通过少量问题完成测试，最后获得自己的游戏人格类型与行为分析。

产品目标：
- 快速吸引用户开始测试
- 用游戏入口增强代入感
- 用简短问答降低答题负担
- 用有仪式感的结果页提升完成价值

页面流程：
首页 → 选择游戏 → 测试答题 → 结果页

页面风格：
- 深色科技感
- 玻璃拟态卡片
- 紫蓝渐变氛围光
- 游戏海报式选择卡片
- 高级、克制、沉浸

请分别输出这 4 个页面的原型：
1. 首页：品牌标题、大按钮、产品卖点卡片
2. 选择游戏页：2~4 个竖版海报式游戏入口卡片
3. 测试页：进度条 + 单题卡片 + 继续按钮
4. 结果页：人格类型标题 + 结果说明 + 操作按钮

要求：
- 每个页面都要有明确视觉焦点
- 流程感清晰
- 交互结构简单
- 页面之间风格统一
